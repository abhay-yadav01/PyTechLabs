# Python Data Type MCQs - Hard Level

### 1. What will the following code output?
```python
x = [1, 2, 3]
y = x
y.append(4)
print(x)
```
A) `[1, 2, 3]`  
B) `[1, 2, 3, 4]`  
C) `None`  
D) `Error`

**Answer:** B

### 3. What is the result of the following?
```python
x = [1, 2, 3]
print(x[3])
```
A) `IndexError`  
B) `None`

D) `Error`

**Answer:** A

### 4. What is the output of the following code?
```python

print(x)
```

### 5. What is the result of the following?
